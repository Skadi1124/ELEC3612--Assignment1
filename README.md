# ELEC3612 Assignment 1

This repository contains the implementation of supervised, unsupervised, and ensemble machine learning models using the Bank Churn dataset.

## Models and Methods

The notebook includes:

- Data preprocessing
- Logistic Regression, Support Vector Machine (SVM), Decision Tree, Random Forest
- Model evaluation using Accuracy, Precision, Recall, F1-score, and Confusion Matrix
- Hyperparameter tuning using GridSearchCV
- Ensemble learning using Voting Classifier
- K-means clustering for unsupervised learning

## Dataset

Churn_Modelling.csv

The dataset contains customer information such as age, gender, account balance, and activity status, and is used to predict whether a customer will leave the bank.

## How to Run

1. Make sure `Churn_Modelling.csv` is in the same folder as the notebook  
2. Open `ELEC3612.ipynb` in Jupyter or Google Colab  
3. Run all cells
